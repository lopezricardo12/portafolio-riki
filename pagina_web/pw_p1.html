Participación 1: Partes de una página web

En este trabajo aprendí cuáles son las partes principales de una página web y la función que tiene cada una. Comprendí que el encabezado incluye el logo y el menú de navegación, que ayudan a identificar el sitio y a moverse entre sus secciones. También entendí que las imágenes hacen más llamativa la página y que el contenido es la información más importante para los usuarios. Además, conocí que la página de inicio es la primera que aparece al ingresar, mientras que el pie de página contiene datos de contacto y redes sociales. Aprendí que los CTA sirven para invitar a realizar acciones, el blog permite compartir información y los formularios ayudan a recopilar datos. Finalmente, realicé una maqueta para observar la ubicación de cada elemento dentro de una página web

Tarea 1: Selectores en CSS

En este trabajo comprendí el funcionamiento de los selectores en CSS y su importancia dentro del diseño de páginas web. Aprendí que existen diferentes tipos de selectores que permiten aplicar estilos de forma precisa a los elementos HTML, dependiendo de su etiqueta, clase, identificador o relación con otros elementos. También realicé ejemplos prácticos que me ayudaron a reforzar el conocimiento y a entender cómo se utilizan en la creación de sitios web.
