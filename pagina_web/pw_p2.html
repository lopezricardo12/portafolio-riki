Tarea 2: Maquetación de una página web

En esta actividad pegué los códigos de mis archivos HTML y CSS en un documento en blanco y realicé capturas de pantalla de la maquetación en el navegador para comprobar su correcto funcionamiento. La estructura de la página incluye un encabezado dividido en tres secciones, un menú de navegación, un Main con tres partes y un pie de página. Esto me permitió comprender mejor la organización de una página web de manera ordenada. Aprendí a trabajar con HTML y CSS para estructurar una página web y distribuir adecuadamente sus elementos.

Tarea 3: Propiedades de CSS para el Menú de navegación.

En esta actividad analicé un archivo de Classroom sobre las propiedades de CSS e identifiqué aquellas que ayudan a mejorar el diseño del menú de navegación. Además, copié mis códigos HTML y CSS en un documento y realicé capturas de pantalla de mi página para comprobar la aplicación de los estilos. Aprendí a utilizar CSS para optimizar la apariencia y organización del menú de navegación en una página
